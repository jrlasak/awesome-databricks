## Getting Started

New to Databricks or looking for a structured path? Start here.

- **[freeCodeCamp Databricks Course](https://www.youtube.com/watch?v=0Hd5vYqin7w)** - Free 7.5-hour video course for DE Associate certification.
- **[The Databricks 100](https://github.com/jrlasak/databricks-100)** - 100 must-know concepts for data engineers. 10 categories x 10 concepts with difficulty ratings, certification mapping, and a 100-day challenge.
- **[Databricks DE Associate Certification Lab](https://github.com/jrlasak/databricks_data_engineer_associate_cert_prep)** - Free hands-on lab that runs on Databricks Free Edition. 4-6 hours, covers the full exam scope.
- **[Databricks Academy](https://customer-academy.databricks.com/learn)** - Official free self-paced courses for every certification path.
- **[DataDojo](https://dojo.dataengineer.wiki)** - Daily Databricks practice exercises. 633+ questions with streaks and progress tracking.

## Learning

### Certifications

All exams: $200 USD, multiple choice, proctored online. Databricks does not publish official passing scores (community-reported ~70%). [Full catalog](https://www.databricks.com/learn/training/certification).

| Exam                                                                                                                         | Level        | Questions | Time    |
| ---------------------------------------------------------------------------------------------------------------------------- | ------------ | --------- | ------- |
| [Data Engineer Associate](https://www.databricks.com/learn/training/certification/data-engineer-associate)                   | Associate    | 45        | 90 min  |
| [Data Engineer Professional](https://www.databricks.com/learn/training/certification/data-engineer-professional)             | Professional | 59        | 120 min |
| [Machine Learning Associate](https://www.databricks.com/learn/training/certification/machine-learning-associate)             | Associate    | 48        | 90 min  |
| [Machine Learning Professional](https://www.databricks.com/learn/training/certification/machine-learning-professional)       | Professional | 59        | 120 min |
| [Generative AI Engineer Associate](https://www.databricks.com/learn/training/certification/genai-engineer-associate)         | Associate    | 45        | 90 min  |
| [Data Analyst Associate](https://www.databricks.com/learn/training/certification/data-analyst-associate)                     | Associate    | 45        | 90 min  |
| [Apache Spark Developer Associate](https://www.databricks.com/learn/training/certification/apache-spark-developer-associate) | Associate    | 45        | 90 min  |

Free accreditations (for customers/partners): Databricks Fundamentals, Generative AI Fundamentals, AI Agent Fundamentals, Platform Administrator, Cloud Platform Architect.

## Open Source & Tools

The open-source projects that power the Databricks platform.

### Open Source

- [delta](https://github.com/delta-io/delta) - Open-source storage framework enabling Lakehouse architecture. The storage layer under every Databricks table. 8.7K stars.
- [Apache Spark](https://github.com/apache/spark) - Unified analytics engine powering Databricks. 43.1K stars.
- [MLflow](https://github.com/mlflow/mlflow) - Experiment tracking, model registry, and deployment. Built into Databricks. 25.1K stars.
- [unitycatalog](https://github.com/unitycatalog/unitycatalog) - Open-source Unity Catalog. Multi-modal data & AI catalog supporting Delta, Iceberg, Hudi. 3.3K stars.
- [delta-rs](https://github.com/delta-io/delta-rs) - Read/write Delta tables from Python without Spark. Useful for lightweight scripts and CI/CD. 3.2K stars.

### Data Quality

- [great_expectations](https://github.com/great-expectations/great_expectations) - Data quality framework with 50+ built-in checks. Supports Spark/Databricks. 11.3K stars.
- [soda-core](https://github.com/sodadata/soda-core) - Define data quality checks in YAML, run against your tables. 2.3K stars.
- [dbt-databricks](https://github.com/databricks/dbt-databricks) - Official dbt adapter. SQL and Python models on Databricks. 341 stars.

### Examples & Demos

- [dbdemos](https://github.com/databricks-demos/dbdemos) - One-line installer for end-to-end Lakehouse demos (DLT, CDC, MLOps, GenAI). Try before you build. 410 stars. By Databricks.
- [ai-dev-kit](https://github.com/databricks-solutions/ai-dev-kit) - Toolkit for AI coding agents (Cursor, Claude Code) to write Spark pipelines and jobs. 1.1K stars. By Databricks.
- [bundle-examples](https://github.com/databricks/bundle-examples) - Official Asset Bundle examples for jobs, pipelines, and ML projects. 308 stars. By Databricks.
